Project01 - Password Checker

Due: Mon, Feb 14, 2022 at 11:59 PM to Github Classroom Assignment

Background

Passwords are a fundamental part of computer security. In this assignment, we will explore some common ways to attack passwords, and techniques for measuring the randomness (entropy) of a password.

Requirements

  1. You will develop a C program which checks a given password (at most 64 characters) against a list of common passwords:
    1. Is the password on a list of 10,000 common passwords?
    2. Is the password one of those passwords, after substituting common “l33t speak” transformations like ‘e’ to ‘3’
    3. Is the password one of those passwords, including a ‘1’ at the end?
  2. Your program will calculate the number of bits of entropy for the given password, using the algorithm given below.
  3. Your program will be built by a Makefile you provide.

Given

  1. The GitHub Classroom Assignment contains a file called passwords.c which defines the list of 10,000 common passwords
  2. Although there are a large number of l33t substitutions, we will use these common ones:
    1. Change ‘0’ to ‘o’
    2. Change ‘3’ to ‘e’
    3. Change ‘!’ to ‘i’
    4. Change ‘@’ to ‘a’
    5. Change ‘#’ to ‘h’
    6. Change ‘$’ to ‘s’
    7. Change ‘+’ to ‘t’
  3. The test case for plus1 is separate from the test case for l33t, so yankees1 should match yankees, but y@nk33s1 should not match yankees.
  4. The number of bits of entropy is given by the equation: log2(number_of_possible_characters ^ length_of_password)
  5. To calculate the number of possible characters, you should loop over the password, increasing the number as shown:
    1. Add 26 if a lower case character is present
    2. Add 26 if an upper case character is present
    3. Add 10 if a digit is present
    4. Add 32 if a printable (use C isprint() from ctype.h) is present but is not upper case, lower case, or a digit
  6. log2(n) (log base 2) is given by log(n) / log(2)
  7. In order to use the C log() function, you’ll need to #include <math.h> and link the executable with gcc -lm
