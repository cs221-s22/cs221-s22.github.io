Project02 - Hardware Store Inventory

Due: Tue, Mar 1, 2022 at 11:59 PM to Github Classroom Assignment

Requirements

  1. You will write a C program and Makefile to help manage stocking parts at a hardware store
  2. Your program will:
    1. Read a CSV file given on the command line
    2. Build a linked list of inventory structures
    3. Sort the list of inventory structures ascending by the aisle in the store where the item may be found.
    4. You may sort the list as you insert items, or use Insertion Sort after the list is built. You must not use C qsort()
    5. Output the sorted list exactly as shown below.
  3. You must scan the characters yourself, without using strtok()

Given

  1. We will demonstrate linked lists and discuss Insertion Sort.
